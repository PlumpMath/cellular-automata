* What it is

Clojure implementations of some [[http://en.wikipedia.org/wiki/Cellular_automata][Cellular Automata]], specifically Brians
Brain and Conway's Game of Life.

* How to use it

To start it in your REPL:
#+BEGIN_SRC clojure
> (use 'cellular-automata)
> (start-world)
#+END_SRC

To stop it:
#+BEGIN_SRC clojure
> (stop-world)
#+END_SRC
